Aspect Ratio Images

Combining the padding "hack" with css variables, we can make a wrapper for any image, to display at a certain size. All with CSS, no javascript involved.

To make as reusable as possible, we're passing through through the required aspect ratio inline eg. style="--aspect-ratio: 16/9" which sets the padding-top using calc. The image inside the wrapper, is then set to fill the div using object-fit. In this demo it is set to user cover, but you could use a different fill option (fill, contain, cover, none, scale-down).

All images are supplier square to showcase the aspect ratio in effect
